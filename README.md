# СайтСотика

Этот сайт создан для представления контента Сотика — современного, минималистичного и стильного.

## Особенности

- Чистый чёрно-золотой дизайн
- Адаптивный интерфейс
- Плавные анимации
- Фиксированный хедер
- Поддержка мобильных устройств

## Технологии

- HTML5
- CSS3 (с адаптивностью)
- JavaScript (плавные эффекты)

## Как запустить локально

1. Склонируйте репозиторий:
   ```bash
git clone https://github.com/sotikdima1975-star/sotikdimasitebest.git
   ```
2. Откройте `index/index.html` в браузере
