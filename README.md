## Особенности реализации

1. **Восстановлен хэдер**:
   - Фиксированное положение в верхней части страницы
   - Логотип "СайтСотика"
   - Навигационное меню с 6 пунктами
   - Адаптивный дизайн
   - Современные эффекты (backdrop-filter)

2. **Первый экран "Приветствие"**:
   - Размещён под хэдером
   - Занимает 100% высоты окна (за вычетом высоты хэдера)
   - Сохранены все UI-элементы:
     - Левый и правый UI-блоки с линиями и точками
     - Центральное ядро с заголовком и кнопкой
     - Нижняя панель статуса

3. **Соблюдение ТЗ**:
   - Строгая реализация требований
   - Минималистичный, системный стиль
   - Отказ от декоративных элементов
   - Использование только разрешённых цветов и шрифтов

## Технические требования

- Только HTML, CSS, JavaScript (без внешних библиотек)
- Адаптивность для мобильных устройств
- Высокая производительность
- Доступность
