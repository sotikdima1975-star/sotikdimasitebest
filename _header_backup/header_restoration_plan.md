# План восстановления хэдера

## Состояние на 2026-02-11

- Хэдер отсутствует на всех страницах сайта
- Меню полностью удалено
- Навигация по страницам нарушена
- Структура страницы нарушена

## Источники для восстановления

1. Техническое задание (TECHNICAL_SPEC.md) - содержит требования к интерфейсу
2. Архивные копии (если доступны)
3. Резервные копии системы контроля версий

## Требования к восстановлению

### 1. Хэдер должен содержать:
- Логотип/название сайта
- Навигационное меню
- Все пункты меню:
  - Главная
  - О нас
  - Команда
  - Игры
  - Стрим
  - Партнёры
- Рабочие ссылки на соответствующие разделы
- Адаптивность для мобильных устройств
- Соответствие стилю сайта (чёрный фон, строгий шрифт, минимализм)

### 2. Расположение
- Хэдер должен быть зафиксирован в верхней части страницы
- Первый экран должен располагаться под хэдером
- Высота первого экрана = 100vh - высота хэдера

### 3. Запрещено
- Удалять хэдер
- Изменять структуру меню
- Менять названия пунктов меню
- Менять ссылки навигации
- Скрывать элементы хэдера

## План действий

1. Создать структуру хэдера в index.html
2. Добавить стили для хэдера
3. Проверить работу навигации
4. Убедиться, что первый экран правильно расположен под хэдером
5. Протестировать на разных разрешениях

## Ожидаемый результат

- Полностью восстановленный хэдер
- Работающее меню
- Корректная навигация
- Первый экран под хэдером
- Сохранение стиля сайта
